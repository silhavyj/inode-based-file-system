◆ getDestinationDirectoryPath()

std::string FileSystem::getDestinationDirectoryPath ( std::string  destination)
private

Returns a destination directory form the path given as a parameter.

As the user enters a path, there are several ways of what they might want to do. For example, if the specifies only a target file, the method will return the current directory as it assumes the user wants to work within the current location. In all other cases, the method will cut off the target file and return the remaining string as a path to the file. For example, 'cp dat/file.txt /movies/file2.txt' will return '/movies/'. This method is used when copying/moving withing the file system as well as importing/exporting files.

Parameters
destinationdestination path e.g. 'doc/zos/file.txt'
Returns
the destination folder e.g. 'doc/zos/'

◆ getDestinationFileName()

std::string FileSystem::getDestinationFileName ( std::string  source,
std::string  destination 
)
private

Returns a destination file name from the path given as a parameter.

As the user enters a path, there are several ways of what they might want to do. For example, they can specify the target file name directly e.g. 'file.txt' or 'doc/file.txt'. In either of these cases, the method will return 'file.txt'. Another case is when the user enters only a target directory. For example, 'incp file.txt doc/zos/'. In this case, the target file name is the same as the original file - 'file.txt'. This method is used when copying/moving files within the file system as well as importing/exporting files.

Parameters
sourcesource file e.g. /doc/source.txt
destinationwhich can be either a file or only a directory
Returns
the destination file name
